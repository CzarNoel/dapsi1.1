PROJETO

A cultura da saúde mental começa pelo conhecimento, o qual traduziremos em insights através do levantamento de dados a partir de pesquisas internas às organizações.

Assim, futuras análises de dados e consequentes tomadas de decisões pelos gestores serão facilitadas, inclusive a humanização laboral.

 
 

As principais bases:

  • Psicologia Organizacional
  • Ética
  • Analise de Dados

As principais ferramentas:

  • Formulários de pesquisa que geram banco de dados tabelados
  • Jupyter Notebook para a análise de dados
  • Relatórios facilmente compartilháveis com gestores

As principais metas:

  • Viabilizar economicamente a análise de dados para Psicólogos Organizacionais
  • Criar insights poderosos
  • Promover a cultura da saúde mental dentro das organizações

SOBRE

Nascemos com a intenção de trazer a tecnologia ao serviço da Psicologia Organizacional, utilizando fundamentos da análise de dados e da ética da POT (Psicologia Organizacional e do Trabalho) e do CFP (Conselho Federal de Psicologia) dentro das organizações.
